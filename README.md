## 👁️ The Vision

**Ideator** is not a note-taking app. It is not a whiteboarding tool. It is a **metacognitive environment** designed to break linear thinking. 

Traditional tools force ideas into rigid, top-down structures (documents, lists, chat interfaces). Human creativity is inherently spatial, chaotic, and infinite. Ideator maps directly to how the brain actually works by providing an unbounded dark-mode canvas powered by ultra-fluid micro-interactions.

Stop outlining. Start mapping.

---

## ⚡ God-Mode Aesthetics & Architecture

We didn't just build a canvas; we engineered an *experience*. Every pixel is tuned for flow state.

- **Infinite Spatial Mapping**: Powered by a heavily customized [Tldraw](https://tldraw.dev/) engine. Infinite pan, infinite zoom, zero constraints.
- **Glassmorphic UI**: Translucent, blurred control panels that hover weightlessly over your ideas.
- **Radial Mouse-Tracking**: Controls react organically to your cursor position, calculating coordinates in real-time to cast subtle volumetric glows.
- **Liquid Particle Physics**: Click interactions spawn bespoke, physics-driven particle bursts, making the UI feel tactile and alive.
- **Vite & React 19**: Lightning-fast HMR and render cycles. Zero latency between thought and execution.

## 🚀 Quick Start (Local Deployment)

Clone the matrix and run it on your own machine.

```bash
# 1. Clone the repository
git clone https://github.com/Khadeer0210/ideator.git

# 2. Enter the directory
cd ideator

# 3. Install dependencies
npm install

# 4. Ignite the dev server
npm run dev
```
*The canvas will open at `http://localhost:5173`.*
